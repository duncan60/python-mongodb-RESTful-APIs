# python-mongodb-RESTful-APIs
practice RESTful APIs to python and mongodb 


## Setup
```
pip install virtualenv
virtualenv env
. env/bin/activate
pip install -r requirements.txt  
```
mongodb db format and use db
```
use bookList
insert data format: {
  title: '',
  price: '',
  date_updated:'',
  date_created:''
}
```
